# SMS-Spam-Detector

In this repo we used the dataset from <a href="https://archive.ics.uci.edu/ml/datasets/sms+spam+collection">UCI</a>.

Use encoding = "Windows-1252" with pd.read_csv

To run the app, in a terminal :

streamlit run app.py
